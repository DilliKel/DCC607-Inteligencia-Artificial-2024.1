# Projeto Final 2024.1 — PLN e Sistema de Recomendação de Livros

**Grupo:** Kelvin Araújo Ferreira, Guilherme Lucas Pereira Bernardo, Mateus Moraes de Moura.

## Problema

Sistema de recomendação de livros baseado em Processamento de Linguagem Natural (PLN): a partir de um livro lido e do motivo de o usuário ter gostado dele, o sistema sugere outros livros com temas ou autores similares (`dataset-livros.csv`).

## Arquivos

- `frente-2-pln.ipynb` — notebook do Google Colab com a exploração do dataset e o processamento de linguagem natural para gerar as recomendações.
- `dataset-livros.csv` — dataset com colunas `Livro Lido`, `Motivo do Gostar` e `Livros Recomendados`.
- `enunciado.pdf` — descrição original do trabalho final da disciplina.
