# Projeto Final 2023.1 — Otimização por Colônia de Formigas (ACO)

**Grupo GHK:** Guilherme Lucas Pereira Bernardo, Hugo Lima Romão, Kelvin Araújo Ferreira.

O grupo explorou três frentes possíveis para o projeto final — algoritmos genéticos, aprendizagem de máquina e otimização por colônia de formigas — e escolheu a **Frente 03 (Colônia de Formigas)** como entrega final.

## Problema

Simular um algoritmo de otimização por colônia de formigas (ACO) para encontrar o menor caminho em um grafo, inspirado no comportamento de formigas reais depositando feromônio nas trilhas mais curtas até a comida.

## Arquivos

- `aco-4-vertices.ipynb` — versão de aula com um grafo simples de 4 vértices, usada como base de aprendizado do algoritmo (sem erros de execução).
- `aco-5-vertices-final.py` — versão final do grupo, com um grafo de 5 vértices (`Frente03-Final.py` original).
- `pseudocodigo-aco.md` — pseudocódigo geral do algoritmo ACO usado como referência.
- `grafos/` — imagens dos grafos usados nos testes (`grafo-1.png`, `grafo-2.png`, `grafos-ia-aco.png`).
- `aco-5-vertices-rascunhos/` — rascunhos e tentativas intermediárias até chegar à versão final (mantidos para referência do processo, sem exclusões).
